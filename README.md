# @gennadysx/react-native-pincher
Pinch any view

supports **TurboModules**.
****


## Installation

```sh
npm install @gennadysx/react-native-pincher
```

## Usage

```js
import PincherView from '@gennadysx/react-native-pincher';

<PincherView>
  <Image source={...}> // or any other view
</PincherView>
```
